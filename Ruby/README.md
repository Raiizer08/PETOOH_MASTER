### DESCRIPTION

PETOOH interpreter written in [Ruby](http://ruby-lang.org).

### DEPENDS

[Ruby](http://ruby-lang.org) 1.9.3 or higer.

### USAGE

To run:

    ~$ ruby petooh.rb file

To get some help:

    ~$ ruby petooh.rb -h
